## Getting Started

To get a local copy up and running follow these simple example steps.

```sh
git clone https://github.com/algerina/AI-summarizer.git
```

Then

```sh
cd AI-summarizer
```

To install packages

```sh
npm install
```

To Run the app

```sh
npm start
```
